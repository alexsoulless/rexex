# REXEX (Remote Execution)

## Описание проекта

REXEX - учебный проект, который был выполнен в рамках курсовой работы. Он позволяет удалённо выполнять код на удалённом ПК. Это может быть полезно при нехватке локальных вычислительных ресурсов для решения определённых задач.


## Основные возможности

- Выполнение программ на удаленном ПК
- Разгрузка локальных ресурсов устройства
- Использование более производительного компьютера независимо от местоположения пользователя

## Технологии

Проект REXEX реализован с использованием следующих технологий:

- PyQt6: для создания удобного графического интерфейса пользователя
- Docker: для обеспечения изоляции и портативности выполняемых задач
